HTTP Client-Server Project — загрузка файлов по ключевым словам
Что делает проект
Это клиент-серверное приложение, которое позволяет:

Ввести ключевое слово (например, в данном случае как пример использовались только два ключевых слова -  птицы или природа);

Получить список URL (ссылок на изображения);

Скачать выбранный файл через сервер;

Сохранить файл на клиенте в LocalStorage;

Просматривать скачанные файлы даже без интернета (оффлайн).

Что реализовано
Серверная часть (Node.js + Express)
Хранит ключевые слова и связанные с ними ссылки в файле keywords.json;

Обрабатывает запросы от клиента:

Возвращает список ссылок по ключевому слову;

Скачивает указанный файл и сохраняет его;

Отправляет прогресс загрузки и имя файла клиенту.

Клиентская часть (HTML + JavaScript)
Отправляет ключевое слово на сервер;

Отображает список найденных URL;

Позволяет выбрать и скачать файл;

Показывает прогресс загрузки;

Сохраняет файл в LocalStorage;

Показывает список загруженных файлов;

Позволяет открыть файлы без подключения к серверу (из памяти браузера).

Как проверить проект
Склонировать репозиторий:

bash

git clone https://github.com/Gpancerov/http-client-offline-viewer.git
cd http-client-offline-viewer
Установить зависимости и запустить сервер:

nginx

npm install
node server.js
Открыть клиент в браузере:

Открыть файл index.html двойным кликом или через локальный веб-сервер (например, Live Server в VS Code).

Протестировать:

Ввести ключевое слово (птицы или природа);

Выбрать ссылку и нажать кнопку "Скачать";

После загрузки нажать "Показать" — файл откроется даже в оффлайне.

Заключение
Приложение реализует клиент-серверное взаимодействие, загрузку данных из интернета по ключевым словам и оффлайн-доступ к этим данным. Проект подходит для проверки работы с JavaScript, Node.js и основами веб-разработки.
